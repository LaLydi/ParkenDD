ParkenDD for iOS

ParkenDD shows you a list of the number of available public parking spots in Dresden, Germany. This information is taken indirectly from the official page at dresden.de.

If you're using Android have a look at the original ParkenDD from Johannes Kliemann.

Check out RobTranquillo/parkplaetzeDD and jklmnn/ParkenDD-Server for the server files that scrape the official site and serve the pretty JSON this app runs on.
